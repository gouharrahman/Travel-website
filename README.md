# Northern Cyprus Travel Website

A full-stack travel and booking website developed for 
Internet Programming course (COMP464) at the European 
University of Lefke (2022).

## Features
- Destination browsing (Girne, Famagusta, Lefkosha, Lefke)
- Hotel/service booking system
- User signup and authentication
- Feedback system
- Responsive design

## Technologies
HTML | CSS | PHP | MySQL | JavaScript
